## Backgrounds (PNG/SVG)

Dépose ici les fichiers utilisés comme **fond d’étiquette**.

Noms attendus par défaut par `Outil gestion étiquettes Claude.html`:

- `plat-8cm.png` (fond rond Ø8cm)
- `tapas-6cm.png` (fond rond Ø6cm)
- `rect-90x55.png` (fond rectangulaire 90×55mm)

Notes:
- PNG est le plus simple et fiable pour `html2canvas`.
- Si tu utilises du SVG, adapte les chemins dans la constante `BACKGROUNDS` du HTML.
